xkcd API Version 2.0

I needed to be able to json fetch all of the xkcd projects for a mobile app. As the xkcd API exists on xkcd you could only get either the latest comic or you could get a comic by the id number. For my mobile app I soon realized that I would have to make 1,880 network requests to get all of the comics so I wanted to build something where I could just get them all. This API gets the latest comics every 6 hours so there is a posibility that you might need to use another method to get the very newest ones.
